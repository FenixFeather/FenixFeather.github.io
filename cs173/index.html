  • [X] Big O
    • [X] Define what it means for a function f to be asymptotically smaller than another function.
      • \(f(n) << g(n)\) if and only if \(\lim_{n \rightarrow \infty}\frac{f(n)}{g(n)} = 0\)
    • [X] For specific functions f and g, identify whether f is asymptotically smaller than g.
      • We only apply the above definition to straightforward functions.
    • [X] Know the asymptotic relationships among key primitive funcitons: constant, log(n), n, nlogn, polynomials of higher orders, exponentials, factorial.
      • 1 << n << n log n << n2 << 2n << n!
    • [X] Given functions f and g, prove that f is O(g)
      • f(n) is only O(g(n)) if and only if there are positive real numbers c and k such that \( 0 \leq f(n) \leq cg(n)\) for every \(n \geq k\)
        • Find numbers for this. Overkill is acceptable.x
      • If f(n) is O(g(n)) and g(n) is O(f(n)), then f is \(\Theta(n)\).
